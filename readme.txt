Description
The plugin adds a HTML snippet of Cookie on a Web Page.

Installation
Upload the plugin files to the /wp-content/plugins/site-cookie-setting directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.


Usage
Go to the 'scs page setting menu' in the WordPress dashboard.
Add Location of cookie accepter HTMl where you wanted to show, add the Cookie message you want to show, choose button color.
Click on 'Save setting.
The HTML snippet will be added to the specified location on the Web page.
Note: You must set the Privacy Policy page from Setting>Privacy and choose Privacy Policy page option from dropdown menu. If you don't have Privacy Policy page, you must create one.
